# Registro-civil
Somos un grupo de jovenes desarrolladores en los cuales colaboramos DIDIPROGRAM,JOTABARBOZA, MIRAGIOOM y yo. 

Simulamos un sistema de REGISTRO-CIVIL desarrollado con Python usando SQLite, tomando en cuenta todas las funcionalidades necesarias que deberia tener un sistema de registro civil, donde se pueden generar reportes de: Nacimientos, Matrimonios, Divorcios, Defunciones y tambien permite el registro de Ciudadanos en nuevas areas, registrar matrimonios, nacimientos, defunciones y divorcios. 

Este proyecto se hizo con la funcionalidad de aprender a colaborar en grupo y poder solventar con pensamiento critico y objetivo los diferentes retos que se puedan generar a la hora de desarrollar o crear una API, al igual que simular un ambiente colaborativo y empresarial en donde se delegan tareas, funciones y realizamos reuniones con metodologias Agiles como SCRUM para poder llevar un control del progreso de las tareas asignados o dar apoyo.
